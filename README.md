## Tech Stack
- [React.js](https://reactjs.org)
- [Jest](https://jestjs.io)

## Getting Started

### Clone the project
```
git clone https://github.com/joshuadavidang/jest-js.git
```

### Install dependencies
```
npm install
```

### To run React locally
```
npm run start
```

### To run React testing library
```
npm run test
```
